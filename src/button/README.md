### Usage

**Flat**

```
<Button>
  Flat Button
</Button>
```

**Colored**

```
<Button accent>
  Colored button
</Button>
```

**Raised**

```
<Button raised>
  Raised button
</Button>
```

**Disabled**

```
<Button raised disabled>
  Raised disabled button
</Button>
```
